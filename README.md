# SafeTrail — AI Travel Planner 🌍✈️

SafeTrail is an AI-powered travel planner that generates personalized, safe, and optimized travel itineraries using multi-agent AI systems.

## 🚀 Features
- AI-generated travel itineraries
- Safety analysis using AI guardrails
- Real-time travel data (Tavily API)
- User authentication (JWT)
- Trip history tracking

## 🧱 Tech Stack
- Frontend: React + Tailwind (Vite)
- Backend: FastAPI (Python)
- Database: MongoDB Atlas
- AI: LangGraph multi-agent system
- LLM: Groq API
